# gigastock

gigastock is a simple user freindly webapp/website to cheack gag2 (Grow A Garden 2) stocks 

## Run With Live Updates (important)

If you open `index.html` directly with `file://`, the official live API can return `403` and seeds will fall back to predictor values.

Run the local server instead:

```bash
node dev-server.js
```

Then open:

```text
http://127.0.0.1:8000/
```

This serves the app and proxies `/api/gag2-stock.json` to the official GAG2 wiki stock API.
